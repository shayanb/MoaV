11. DNS Tunnel

Last Resort
Port 53/udp

Very slow but works when everything else is blocked. Tunnels data through DNS queries. Only use as last resort.

How DNS Tunnel Works

DNS tunneling encodes data in DNS queries. Since DNS is essential for the internet, it's rarely completely blocked. The trade-off is very slow speeds (expect 10-50 KB/s).

Desktop Setup (Windows/macOS/Linux)

  1. Download dnstt-client for your OS
  2. Open Terminal/Command Prompt
  3. Run this command to create a local SOCKS5 proxy on port 1080:
dnstt-client -doh https://1.1.1.1/dns-query -pubkey {{DNSTT_PUBKEY}} {{DNSTT_DOMAIN}} 127.0.0.1:1080

Then configure your browser to use SOCKS5 proxy: 127.0.0.1:1080

Mobile Setup (Android)

HTTP Injector supports DNS tunneling:

  1. Install HTTP Injector from Play Store
  2. Create new config with DNS Tunnel mode
  3. Enter DNS domain: {{DNSTT_DOMAIN}}
  4. Set remote proxy to your server

Note: iOS has no good DNS tunnel apps. Use desktop if possible.

When to use Tor

تونل DNS چگونه کار می‌کند

تونل DNS داده‌ها را در درخواست‌های DNS کدگذاری می‌کند. چون DNS برای اینترنت ضروری است، به ندرت کاملاً مسدود می‌شود. سرعت بسیار کم است (انتظار ۱۰-۵۰ کیلوبایت در ثانیه).

  3. این دستور را اجرا کنید:
dnstt-client -doh https://1.1.1.1/dns-query -pubkey {{DNSTT_PUBKEY}} {{DNSTT_DOMAIN}} 127.0.0.1:1080

سپس مرورگر را برای استفاده از پروکسی SOCKS5 تنظیم کنید: 127.0.0.1:1080
